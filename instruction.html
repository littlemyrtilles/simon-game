Instructions

  1. Press any key to start the game. Turn on the volume to hear sounds.

  2. A color will flash. Repeat by pressing the same color.

  3. A second color will flash. Now press the previous color and the new color button.

  4. The game will continue. After each color sequence that you repeat sucessfully, a new color will be added to the end.

  5. When you fail to complete a sequence successfully, the game ends.

  6. Press any key to try again!
